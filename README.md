This contians a datasheet and example code for the ARM-DDHBSD_0_10-0S High Effieciency Dual H-Bridge and Stepper Driver.
The code is specfifically for use with the arduino IDE, 
the H-Bridge code can easily be adapted to any microcontroller.
For users intending to drive full-step stepper motors using an Arduino or ESP32 we have provided a library native to the Arduino IDE, called ARAM_FS_Stepper.
